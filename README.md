# Where in the world.

Una aplicacion web para ver informacion relevante de todas las naciones del mundo.

A la aplicacion se puede acceder a traves del navegador de dispositivos moviles y PCs, debido a que esta diseñado para adaptarse a cualquier tamaño de pantalla y así brindar la mejor experiencia al usuario.

## Dark Mode

En la parte superior derecha se encuentra la opcion "Dark Mode" la cual cambiara los colores de la aplicacion para personalizar la experiencia del usuario.

### Busqueda por coincidencias de nombre

Con la barra de busqueda de texto se puede ingresar el nombre de una nacion del mundo, este nombre puede ser parcial o total, ya que el sistema esta diseñado para buscar coincidencias a medida que el usuario digita en la caja de busqueda.

### Busqueda por region

Con el selector de regiones se pueden filtrar listados de paises que pertenecen a un mismo continente por orden alfabetico.

### Detalles de una nacion

Al realizar click en cualquier tarjeta de una nacion la aplicacion muestra mas datos relevantes de la nacion seleccionada, de esta forma el usuario puede consultar datos adicionales a los que se muestran en el listado de naciones.

Tambien se puede ingresar al detalle de una nacion por medio de la pantalla de detalles. Si la nacion que se esta consultando tiene fronteras con otros paises, estos apareceran listados en el item `borders`. Al dar click sobre cualquiera de las naciones listadas alli se ingresa al detalle de dicha nacion.


## Ejecucion del proyecto

Este proyecto fue creado con [Create React App](https://github.com/facebook/create-react-app).

Para proceder con la ejecucion de la aplicacion se debe instalar previamente las dependencias necesarias para lo cual se ejecuta el comando:

### `npm install`

Una vez finalizada la instalacion de dependencias se procede a compilar la aplicacion, para lo cual se usa el comando:

### `npm run build`

Finalmente, se puede proceder a ejecutar la aplicacion con el comando:

### `npm start`

Para ingresar a la aplicacion escribimos en una ventana en el navegador `http://localhost:8080/` donde la aplicacion quedara interactuable para el usuario.
